For Beginner

  1. list-style이란?

    2. 리스트를 출력할 수 있는 <ol>,<ul>를 꾸며주는 명령어. 리스트를 꾸며줄 수 있는 명령어 속성은 아래 3가지가 있다.

    • list-style-type
    • list-style-position
    • list-style-image

  2. list-style-type출력하기

  • <ol>tag list-style-type출력하기

    1. list-style-type: ﻿decimal

      1. ﻿<body>tag 안에 들어갈 list를 먼저 입력한다. class="li0"을 생성한다. 여러개의 list를 만들 것이기 때문에 class로 묶어주고 수식해준다.

        2. HTML 코드

        <ol class="li0">

        <li>list-style-type: decimal</li>

        <li>list-style-type: decimal</li>

        <li>list-style-type: decimal</li>

        </ol>


      2. <head>태그 안에 <style>태그를 열고 .li0을 수식하는 명령어들을 넣어준다.

        3. CSS 코드

        <head>

        <style>

        .li0{list-style-type: decimal;}

        </style>

        </head>


      3. 저장을 한 후 출력한다.

        4. 출력 화면
        • list-style-type: decimal
        • list-style-type: decimal
        • list-style-type: decimal

    2. list-style-type:decimal-leading-zero

      1. <body>tag 안에 들어갈 list를 먼저 입력한다. class="li1"을 생성한다. 여러개의 list를 만들 것이기 때문에 class로 묶어주고 수식해준다.

        2. HTML 코드

        <ol class="li1">

        <li>list-style-type: decimal-leading-zero</li>

        <li>list-style-type: decimal-leading-zero</li>

        <li>list-style-type: decimal-leading-zero</li>

        </ol>


      2. <head>태그 안에 <style>태그를 열고 .li1을 수식하는 명령어들을 넣어준다.

        3. CSS 코드

        <head>

        <style>

        .li1{list-style-type: decimal-leading-zero;}

        </style>

        </head>


      3. 저장을 한 후 출력한다.

        4. 출력 화면
        • list-style-type:decimal-leading-zero;
        • list-style-type:decimal-leading-zero;
        • list-style-type:decimal-leading-zero;
