For Beginner

  1. ﻿text-align 속성이란?﻿

    2. ﻿text-align 속성은 문단 정렬을 시키는 태그이다. 이전 HTML 포스트 중 <p align="left | right | center | justify">를 사용하여 문단 정렬을 하는 법을 포스팅했는데, 이 포스팅에서도 적었듯 HTML5에서는 <p>의 align 속성이 지원되지 않는다. 그래서 문단 왼쪽, 오른쪽, 가운데, 양쪽정렬을 시키고 싶다면 CSS를 사용하여 적용시켜야 한다. <p>태그에 아무 문단 정렬을 시키지 않으면 자동으로 왼쪽 정렬으로 출력된다.﻿

    이전 포스팅 : https://blog.naver.com/juyoung1704/221630610060

  2. ﻿text-align 속성 출력하기

    1. ﻿출력할 문단 태그인 <p>tag 출력

      2. 코드
      <body>
      <p>text-align은 문단 정렬 속성</p>
      </body>

      출력 화면

      text-align은 문단 정렬 속성


    2. ﻿﻿CSS로 text-align 속성 출력하기

      • ﻿﻿왼쪽 정렬 : text-align: left;

        • 코드
        <style>
        p{
        text-align: left;
        }
        </style>

        출력 화면

        text-align은 문단 정렬 속성


      • ﻿오른쪽 정렬 : text-align: right;﻿

        • 코드
        <style>
        p{
        text-align: right;
        }
        </style>

        출력 화면

        text-align은 문단 정렬 속성


      • ﻿가운데 정렬 : text-align: center;﻿

        • 코드
        <style>
        p{
        text-align: center;
        }
        </style>

        출력 화면

        text-align은 문단 정렬 속성


      • 양쪽 정렬 : text-align: justify;﻿

        • 코드
        <style>
        p{
        text-align: justify;
        }
        </style>

        출력 화면

        text-align은 문단 정렬 속성
