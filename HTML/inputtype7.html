For Beginner

  1. <input type="reset">이란?

    2. ﻿작성되어 있는 모든 value 값을 리셋(초기화) 시키는 reset 버튼을 출력하는 type.

    ﻿하지만, 양식에 reset 버튼이 있으면 사용자가 실수로 클릭하는 경우가 많은데 이 경우 사용자가 불편함을 느끼게 되므로 되도록 피하는 것이 좋다.

  2. ﻿input type="reset" 출력하기

    3. 코드

    <form>

    <input type="reset" value="Reset">

    </form>
