<h1>~<h6> : 제목 태그

  1. <h>태그란?

    2. <h1>~<h6>태그는 heading으로, 제목 태그를 가리킨다. 제목 용도가 아니라 강조하려고 <h>태그를 남발하면 검색엔진에서 제외되고, 가독성이 떨어지니 제목 용도로만 사용하도록 주의해야 한다.

    <h1>~<h6>태그는 1이 가장 크고 6이 가장 작다.

  2. <h>태그 사용하여 출력하기

    1. <h1> tag : 가장 큰 제목

      2. 코드 <h1>제목1</h1>

      출력화면

      제목1

    2. <h2> tag : 2번째로 큰 제목

      3. 코드 <h2>제목2</h2>

      출력화면

      제목2

    3. <h3> tag : 3번째로 큰 제목

      4. 코드 <h3>제목3</h3>

      출력화면

      제목3

    4. <h4> tag : 4번째로 큰 제목

      5. 코드 <h4>제목4</h4>

      출력화면

      제목4

    5. <h5> tag : 5번째로 큰 제목

      6. 코드 <h5>제목5</h5>

      출력화면
      제목5

    6. <h6> tag : 가장 작은 제목

      7. 코드 <h6>제목6</h6>

      출력화면
      제목6
