    2. HTML 문서에서 ● ★ ♥ ◆ 등의 특수문자를 출력하고 싶을 때는, 해당 문자가 아니라 코드를 출력해야 한다. ♥나 ★같은 경우는 <p>태그 안에 써도 문자 그대로 출력되는 경우도 있지만, <, > 같은 부등호는 태그를 열고 닫는 문자라 html에서 특수문자 코드를 입력해서 출력해야 한다. 그렇지 않으면 < >가 들어간 문장이 통으로 출력되지 않거나, css 문법이 갑자기 문단 중간에 적용되어 html 문법이 먹히지 않는 등 문제가 발생할 수 있다.

    아래는 W3schools가 정리해둔 html 특수문자표 목록이다. 나는 홈페이지를 만들면서 <, >에 대한 특수기호를 많이 써서

    &#60;, &#62;를 쓰는게 벌써 손에 익었다. 다들 처음에는 번거롭고 귀찮은 과정일테지만 쓰다보면 자동으로 외워진다. 처음부터 끝까지 외우려고 하지말고, 쓰면서 자주 쓰는 것들은 메모해두고 써보는 것을 추천한다.
