For Beginner

  1. <input type="text">란?

    2. ﻿텍스트를 입력할 수 있는 한 줄의 텍스트 공간을 출력하는 type. placeholder에 텍스트를 입력하면 클릭하기 전까지 텍스트가 보이고, 클릭을 한 후엔 텍스트가 사라지고, 입력한 값만이 쓰여진다.﻿

  2. ﻿input type="text" 출력하기

    • ﻿<label for="﻿text">﻿ID</label>﻿

      • →﻿label : 텍스트 부분을 클릭해도 입력 박스에 커서가 나오게 하는 기능﻿

      →﻿﻿for="text" : 이 <﻿label>태그가 어떤 input 타입에 대한 태그인지 정의해 주는 것. for의 이름과 id의 이름이 같아야 한다.﻿

    • ﻿<﻿input type="text" id="text" name="text" placeholder="text">

      • →﻿id="id이름" : 각 <input type>마다 CSS로 꾸며주기 위해 ID를 사용한다.﻿

      →﻿name="이름" : 값을 제출하고 난 후, 각 값이 어떤 것을 기준으로 한 데이터 값인지 위한 '기준' ﻿

      →﻿placeholder="데이터 입력 전 보여줄 글자" : 홈페이지 로그인을 할 때나 핸드폰 인증 시 값을 입력하지 않았음에도 '무슨 데이터 값'을 입력하면 되는지 알려주는 것. 아래 네이버 로그인 화면에서 클릭하기 전 ID를 입력하는 칸에는 '아이디'를, 비밀번호를 입력하는 칸에 '비밀번호'를 적으라고 안내해 주는 것이 placeholder이다. 주로 회색을 쓴다.﻿




    코드

    <label for="text">ID</label>

    <input type="text" id="text" name="text" placeholder="text">

  3. <﻿input type="password">란?

    4. ﻿비밀번호를 입력할 수 있는 한 줄의 텍스트 공간을 출력하는 type. 텍스트를 입력하면 글자가 입력되지 않고 ●●●●으로 표시된다. placeholder에 텍스트를 입력하면 text타입과 동일하게 텍스트가 출력된다.

  4. ﻿input type="password" 출력하기

    5. 코드

    <label for="password">PASSWORD</label>

    <input type="password" id="password" name="password" placeholder="password">
