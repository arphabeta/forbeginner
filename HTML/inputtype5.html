For Beginner

  1. <input type="button">란?

    2. ﻿클릭이 가능한 버튼을 출력하는 type. 주로 javascript를 사용하여 스크립트를 활성화하는데 사용한다. type="button"은 요소 안에 이미지를 넣을 수 없어 이미지를 넣어 꾸며줄 때는 <button>tag를 사용한다.

  2. input type="button" 코드 풀어보기

    3. <form>

    <input type="button" value="Click me" onclick="msg()">﻿: ﻿Click me 라는 텍스트가 입력되어 있는 버튼을 출력할 것이다. 그리고 클릭을 했을 때(onclick), 메시지(message, msg)를 띄울 것이다.﻿

    </form>


    <script> ﻿ : 이 부분은 javascript 언어를 사용할 것이다.﻿

    function msg() { ﻿ : 메시지(message, msg)를 띄우는 기능을 출력할 것이다.﻿

    alert("Hello women!");﻿: 메시지는 "Hello women!"이라고 출력될 것이다.﻿﻿

    }

    </script>
