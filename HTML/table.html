For Beginner

  1. ﻿﻿<table>, <tr>, <td>, <th> tag 란?

    • <table> : ﻿부모태그. 표를 만들기 위해서 시작과 끝에 무조건 써줘야 하는 태그.﻿

    • <tr> : ﻿줄(행) 태그. 한 행을 열고 닫음을 표시해주는 태그.

    • <td> : ﻿칸(열) 태그. 빈 공간에서 벽면에 충돌할 때마다 <td>tag 를 열고, 닫아준다. 맨 마지막 벽에서는 </td></tr>로 닫아준다.

    • <th> ﻿: <td> tag 자리에 쓰는 태그. <th> tag를 쓰면 표 안의 텍스트가 자동으로 굵게 되고, 가운데 정렬이 된다. ﻿

    ﻿*표는 줄(행)-가로가 중요하다.

    ﻿따라서, 빈 공간에서 시작하여 그리고 싶은 표의 벽면에 충돌할 때마다 <td> tag를 열고 닫아준다.﻿

  2. ﻿기본 2X2 표 출력하기

    3. <table border="1px" width="200px" height="200px"> → 표의 선은 1픽셀 굵기의 표는 가로 200픽셀, 세로 200픽셀로 지정할 것이다.

    <tr><td>A</td><td>B</td></tr> → □□

    <tr><td>C</td><td>D</td></tr> → □□

    </table> → 표를 여기서 닫아줄 것이다.

    코드

    <table border="1px" width="200px" height="200px">

    <tr><td>A</td><td>B</td></tr>

    <tr><td>C</td><td>D</td></tr>

    </table>


    출력 화면
    AB
    CD

  3. ﻿열병합 표 : colspan 사용

    4. ﻿열병합은 <td>tag 안에 colspan 속성을 사용한다. colspan 속성은 columns(기둥)+span(다리건물 따위의 기둥과 기둥 사이)를 뜻한다. colspan="2" 에 들어가는 숫자는 2열을 병합했다는 병합했다는 의미다. 열병합한 칸 숫자가 와야하며, 절대 픽셀을 의미하는 것이 아니다.﻿


    <table border="1px" width="200px" height="200px"> ﻿→ 표의 선은 1픽셀 굵기의 표는 가로 200픽셀, 세로 200픽셀로 지정할 것이다.

    <tr><td colspan="2">A</td></tr> ﻿→ 이 행은 2개의 열을 병합할 것이다. □

    <tr><td>C</td><td>D</td></tr> ﻿→ □□﻿

    </table>



    코드

    <table border="1px" width="200px" height="200px">

    <tr><td colspan="2">A</td></tr>

    <tr><td>C</td><td>D</td></tr>

    </table>



    출력 화면
    A
    BC

  4. ﻿행병합 : rowspan 사용

    5. ﻿행병합은 <td>tag 안에 rowspan속성을 사용한다.rowspan 속성은 row(사람, 사물들이 옆으로 늘어서 있는 열)+span(다리건물 따위의 기둥과 기둥 사이)를 뜻한다. rowspan="2" 에 들어가는 숫자는 2행을 병합했다는 병합했다는 의미다. 행병합한 칸 숫자가 와야하며, 절대 픽셀을 의미하는 것이 아니다.﻿

    ﻿* 행병합의 경우, 첫줄만 잘 보고 작성하면 꼬이지 않는다. 첫줄을 기준으로 하고 표를 작성한다.


    <table border="1px" width="200px" height="200px"> ﻿→ 표의 선은 1픽셀 굵기의 표는 가로 200픽셀, 세로 200픽셀로 지정할 것이다.

    <tr><td rowspan="2">A</td><td>B</td></tr>

    <tr><td>C</td></tr>﻿ → □

    </table>





    코드

    <table border="1px" width="200px" height="200px">

    <tr><td rowspan="2">A</td><td>B</td></tr>

    <tr><td>C</td></tr>

    </table>


    출력 화면
    AB
    C
