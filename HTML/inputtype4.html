For Beginner

  1. <input type="week﻿">란?

    2. ﻿ 날짜를 클릭하면 몇번째 주인지 알려주는 type이 출력된다. 파이어폭스와 사파리에선 지원되지 않는다.

  2. input type="week﻿" 출력하기

    3. 코드

    ﻿<form>

    ﻿<input type="week" id="week" name="week">

    </form>

    출력 화면

  4. input type="time﻿" 출력하기
