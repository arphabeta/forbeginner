  • 다양한 목록 출력하기

    1. <ol>tag 다양한 목록 출력하기

      2. ﻿워드나 구글 문서처럼 자동숫자 메기기가 되는 <ol>태그의 다양한 type

      • <ol type="1">

        • ﻿<ol type="1">은 <ol>태그만 쓴 것과 별다른 차이 없이 출력됩니다. 하지만 문서를 작성하고, 꾸밀 때 문서의 대부분이 a, b, c, d 배열을 하고 있는데 이 목차만 1,2,3,4 순서로 배열을 하고 싶을 경우를 위해 사용되는 속성입니다.

        코드
        <ol type="1">
        <li>목차1</li>
        <li>목차2</li>
        <li>목차3</li>
        <li>목차4</li>
        </ol>

        출력화면
        1. 목차1
        2. 목차2
        3. 목차3
        4. 목차4

      • <ol type="A">

        • ﻿type="A"는 대문자 A, B, C, D 순서로 자동 메김이 설정됩니다.

        코드
        <ol type="A">
        <li>목차1</li>
        <li>목차2</li>
        <li>목차3</li>
        <li>목차4</li>
        </ol>

        출력화면
        1. 목차1
        2. 목차2
        3. 목차3
        4. 목차4

      • <ol type="a">

        • ﻿type="a"는 소문자 a, b, c, d 순서로 자동 매김이 설정됩니다.

        코드
        <ol type="a">
        <li>목차1</li>
        <li>목차2</li>
        <li>목차3</li>
        <li>목차4</li>
        </ol>

        출력화면
        1. 목차1
        2. 목차2
        3. 목차3
        4. 목차4

      • <ol type="I">

        • ﻿type="I"는 Ⅰ, Ⅱ, Ⅲ, Ⅳ 순서로 자동 매김이 설정됩니다.

        코드
        <ol type="I">
        <li>목차1</li>
        <li>목차2</li>
        <li>목차3</li>
        <li>목차4</li>
        </ol>

        출력화면
        1. 목차1
        2. 목차2
        3. 목차3
        4. 목차4

      • <ol type="i">

        • ﻿type="i"는 ⅰ, ⅱ, ⅲ, ⅳ 순서로 자동 매김이 설정됩니다.

        코드
        <ol type="i">
        <li>목차1</li>
        <li>목차2</li>
        <li>목차3</li>
        <li>목차4</li>
        </ol>

        출력화면
        1. 목차1
        2. 목차2
        3. 목차3
        4. 목차4

      • <ol reversed>

        • ﻿<ol reversed>는 기존의 <ol>태그에서 1,2,3,4 출력되던 것들의 순서를 4,3,2,1 순서대로 만드는 속성입니다. reversed라는 단어의 의미처럼 순서를 반대로 하고 싶을 때 사용합니다.

        코드
        <ol reversed>
        <li>목차1</li>
        <li>목차2</li>
        <li>목차3</li>
        <li>목차4</li>
        </ol>

        출력화면
        1. 목차1
        2. 목차2
        3. 목차3
        4. 목차4

    2. <ul>tag 다양한 목록 출력하기

      3. ﻿순서 없이 목록에 ●, ■ 같은 기호로 목차를 표시하는 <ul>tag

      • <ul type="square">

        • ﻿type="square"는 목차를 사각형 (■) 모양의 기호로 출력할 때 사용합니다.

        코드
        <ul type="square">
        <li>목차1</li>
        <li>목차2</li>
        <li>목차3</li>
        <li>목차4</li>
        </ul>

        출력화면
        • 목차1
        • 목차2
        • 목차3
        • 목차4

      • <ul type="circle">

        • ﻿type="circle"은 목차를 속이 빈 원형(○) 모양의 기호로 출력할 때 사용합니다.

        코드
        <ul type="circle">
        <li>목차1</li>
        <li>목차2</li>
        <li>목차3</li>
        <li>목차4</li>
        </ul>

        출력화면
        • 목차1
        • 목차2
        • 목차3
        • 목차4

      • <ul type="disc">

        • ﻿type="disc"는 목차를 속이 채워진 원형(●) 모양의 기호로 출력할 때 사용합니다.

        코드
        <ul type="disc">
        <li>목차1</li>
        <li>목차2</li>
        <li>목차3</li>
        <li>목차4</li>
        </ul>

        출력화면
        • 목차1
        • 목차2
        • 목차3
        • 목차4
