For Beginner

  1. 다양한 폰트 효과

    1. <b>진하게

      2. <b>태그는 bold, 진하게를 의미한다.

    2. <strong>진하게

      3. <strong>태그는 strong, 진하게를 의미한다. <b>태그와 똑같은 기능을 한다.

    3. <i>이탤릭체-기울기

      4. <i>태그는 italics, 이탤릭체. 오른쪽으로 기울어진 글자체를 의미한다.

    4. <small>작은 글자

      5. <small>태그는 small, 작은 글씨를 의미한다.

    5. <big>큰 글자

      6. <big>태그는 big, 큰 글씨를 의미한다.

    6. <u>밑줄

      7. <u>태그는 underline, 밑줄을 의미한다.

    7. <del>취소선

      8. <del>태그는 deleted part, 취소한 부분, 취소선을 의미한다.

  2. 다양한 폰트 출력하기

    1. <b>진하게

      2. 코드 <b>진하게</b>

      출력 화면 진하게

    2. <strong>진하게

      3. 코드 <strong>진하게</strong>

      출력 화면 진하게

    3. <i>이탤릭체-기울기

      4. 코드 <i>이탤릭체-기울기</i>

      출력 화면 이탤릭체-기울기

    4. <small>작은 글자

      5. 코드 <small>작은 글자<small>
      <small><small>작은 글자<small><small>
      <small><small><small>작은 글자<small><small><small>

      출력 화면 작은 글자
      더 작은 글자
      더욱 더 작은 글자

    5. <big>큰 글자

      6. 코드 <big>큰 글자</big>
      <big><big>큰 글자</big></big>
      <big><big><big>큰 글자</big></big></big>

      출력 화면 큰 글자
      더 큰 글자
      더욱 더 큰 글자

    6. <u>밑줄

      7. 코드 <u>밑줄</u>

      출력 화면 밑줄

    7. <del>취소선

      8. 코드 <del>취소선</del>

      출력 화면 취소선
