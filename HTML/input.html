For Beginner

  1. <input>tag란?

    2. <input>tag﻿란 사용자가 직접 데이터나 특정 값을 입력할 수 있는 부분을 출력할 수 있는 태그를 말한다. <input>태그 요소는 비어 있고, 속성 값을 포함해야 웹상에 표시된다. <input>은 inline element (인라인 엘리먼트)이며, 반드시 <form>tag를 부모 태그로 사용해야 한다.

    <﻿label>﻿ : 직접적으로 선택지(옵션, 메뉴 등등)를 클릭하지 않고 텍스트 부분만 클릭해도 옵션 선택이 가능하도록 하는 기능을 부여하는 태그. 마우스 사용자들에게 편리함을 주기 위해 쓰는 태그이다. <button>, <input>, <meter>, <output>, <progress>, <select>, <textarea>tag에 사용한다.﻿

    <input>tag﻿태그는 아래와 같이 이루어져 있다.


    • ﻿<label for="aa">성명</label>﻿

      • →﻿label : 텍스트 부분을 클릭해도 입력 박스에 커서가 나오게 하는 기능﻿

      →﻿﻿for="aa" : 이 <﻿label>태그가 어떤 input 타입에 대한 태그인지 정의해 주는 것. for의 이름과 id의 이름이 같아야 한다.﻿

    • ﻿<input type="type이름" id="id이름" name="이름" placeholder="데이터 입력 전 보여줄 글자">

      • →﻿type="type이름" : 출력하고 싶은 type을 적는 공간. type의 종류에는 text, password, checkbox, date 등이 있다. ﻿

      →﻿id="id이름" : 각 <input type>마다 CSS로 꾸며주기 위해 ID를 사용한다.﻿

      →﻿name="이름" : 값을 제출하고 난 후, 각 값이 어떤 것을 기준으로 한 데이터 값인지 위한 '기준' ﻿

      →﻿placeholder="데이터 입력 전 보여줄 글자" : 홈페이지 로그인을 할 때나 핸드폰 인증 시 값을 입력하지 않았음에도 '무슨 데이터 값'을 입력하면 되는지 알려주는 것. 아래 네이버 로그인 화면에서 클릭하기 전 ID를 입력하는 칸에는 '아이디'를, 비밀번호를 입력하는 칸에 '비밀번호'를 적으라고 안내해 주는 것이 placeholder이다. 주로 회색을 쓴다.﻿
