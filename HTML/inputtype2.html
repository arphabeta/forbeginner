For Beginner

  1. <input type="﻿checkbox">란?

    2. ﻿한가지 혹은 여러 가지를 선택할 수 있는 사각형의 체크박스를 출력하는 type. 체크박스를 클릭하면 '체크' 표시가 나타난다.

  2. ﻿input type="checkbox" 출력하기

    3. 코드 <form>

    <input type="checkbox" name="checkbox1" value="check">check

    <input type="checkbox" name="checkbox2" value="box">box

    <input type="checkbox" name="checkbox3" value="print" checked>print

    </form>

    출력 화면
    check box print

    *한번 클릭해보세요!

  3. <input type="﻿radio">란?

    4. ﻿하나만 선택할 수 있는 원형의 체크박스를 출력하는 type. radio 버튼으로 출력할 항목들의 카테고리인 name값을 모두 동일한 항목으로 입력해야 하며, value는 모두 다른 값을 입력해야 한다. value값은 사용자에게 보이지 않지만 submit 버튼을 사용자가 눌렀을때, 어떤 radio 항목을 선택했는지 보여주는 역할을 한다.

    ﻿웹에 출력했을 때 기본으로 선택되어 있는 radio를 출력하려면 <input>tag의 모든 요소를 입력한 다음 마지막에 checked라고 적으면 된다.

  4. ﻿input type="checkbox" 출력하기

    5. 코드 <form>

    <input type="radio" name="radio" value="radio">radio

    <input type="radio" name="radio" value="button">button

    <input type="radio" name="radio" value="print" checked>print

    </form>

    출력 화면
    radio button print

    *한번 클릭해보세요!
