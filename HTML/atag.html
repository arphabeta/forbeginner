For Beginner
    하이퍼링크 연결형식

    1. a태그로 외부 사이트 링크 삽입하기

      2. ﻿a태그는 하이퍼링크를 삽입하는 태그이다. anchor(닻)의 첫 글자를 딴 a를 사용한다. 배가 정박할 때 사용하는 닻을 의미. 정보의 바다에 정박한다는 시적인 의미를 지니고 있다. -유튜브 생활코딩

      ﻿<a href="첨부하고 싶은 링크" target="_blank" title="링크에 마우스 가져다 댔을 때 미리 보기 할 제목">하이퍼링크 달고 싶은 단어나 문장, 이미지</a>﻿

      입력 코드 <a href="http://www.naver.com" target="_blank" title="네이버">네이버</a>로 이동하기 <a href="http://www.daum.net" target="_blank" title="다음">다음으로 이동하기</a>

      출력 화면 네이버로 이동하기

      다음으로 이동하기

    2. ﻿같은 파일내 HTML 문서에 하이퍼링크 연결하기

      3. ﻿같은 파일에 있는 HTML파일과 연결하고 싶으면 tag 안의 href 속성 안에 HTML 문서명을 치면된다.

      입력 코드 <a href="tag1-기본태그.html" target="_blank" title="기본태그">기본태그</a> <a href="tag2-표만들기.html" target="_blank" title="표만들기">표만들기</a> <a href="tag3-표만들기실습.html" target="_blank" title="표만들기실습">표만들기실습</a>

      출력 화면 기본태그 표만들기 표만들기실습

    3. ﻿이미지에 하이퍼링크 삽입하기

      4. ﻿a태그 사이에서 이미지 태그인 img태그를 사용하면 이미지에 하이퍼링크가 삽입된다.

      입력 코드 <a href="http://www.naver.com" title="네이버"><img src="holiday-4138980_1920.png"></a>

      출력화면

      ﻿
    4. 문서 내의 특정 위치에 하이퍼링크 삽입하기

      5. ﻿문서가 길 경우 하이퍼링크를 클릭하면 문서내의 특정 위치에 도달하도록 하는 태그이다. 하이퍼링크를 클릭하여 도달하게 하고픈 코드 바로 윗줄에 <a name="지정하고 싶은 이름">을 입력해준다. 그 후 하이퍼링크를 달 단어, 문장 혹은 이미지 코드에 부분에 <a href="#name에 지정했던 이름">코드를 작성한다. 꼭 #을 붙여주어야 된다.﻿

      입력 코드 (첫줄) <a name="top"></a> <h1>하이퍼링크 연결형식</h1> (기타 태그들) <ul> <li><a href="#top">맨위로 이동하기</a></li> </ul>

      출력 화면
      • 맨위로 이동하기
